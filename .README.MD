# Premier League Data FetcherA lightweight Linux-based data pipeline that automatically fetches Premier League match data daily from the Football-Data API, stores raw JSON snapshots, and maintains execution logs for monitoring and debugging.---## Project OverviewThis project is a simple but production-style data ingestion pipeline built using Bash scripting.  It demonstrates core Data Engineering concepts such as:- API ingestion- File-based storage (raw layer)- Logging & observability- Idempotent design- Automation readiness (cron-compatible)---## Architecture
API (football-data.org)
↓
fetch.sh (Bash ingestion script)
↓
data/raw/ (raw JSON snapshots)
↓
logs/ (execution logs)
---## Features- Daily automatic data fetching (Premier League matches)- Timestamped raw data storage- Structured logging per execution- Error handling for API/network failures- Idempotent design (prevents duplicate processing)- Backfill-ready architecture---## Project Structure

Key Design Principles
1. Idempotency
The script avoids duplicate processing by checking existing files before fetching.
2. Separation of Layers


Raw data stored separately from logic


Logs isolated from data layer


3. Failure Awareness
All API errors are captured in logs with timestamps and status codes.
