# Premier League Data Fetcher

A Linux-based data pipeline that fetches Premier League match data daily from an external API and stores it as raw JSON files with logging and error handling.
---
1. Project Goal
This project is built to simulate a real-world Data Engineering ingestion pipeline
It demonstrates: 
1: How To Extract data from an API 
2:How To Store low data locally
3: How to track execution using logs
4:How to design a safe, repeatable (idempotent) pipeline
----
## 2. Architecture 
API (Football Data)
↓
Bash Script (fetch.sh)
↓
Raw Data Files (data/raw/)
↓
Logs (logs/)
### Explanation:
- API is the data source
- Bash script is the ingestion layer
- Raw folder stores untouched data
- Logs track what happened during execution
